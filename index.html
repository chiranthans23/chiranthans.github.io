Emergency message classifier using NLP and ML
Developer January 2018 - August 2018

Few thousand Whatsapp messages were used to form a dataset, and was cleaned the dataset to be text-only. Used Tokenisation, POS tagging, and a customized Stemmer to further improve the dataset. A bag of emergency words was formulated, and important features were selected. The undersampled data from the dataset was trained on SVM classifier.

Credit card fraud detection using Machine Learning
Developer August 2017 – May 2018

Used a dataset of bank transactions. Normalized and Feature Engineered the dataset. The undersampled dataset was trained on multiple classifiers and evaluated the model using various statistical analysis parameters. Further, Reinforcement learning was used to track transactions in real-time. Finally, ExtraTrees gave the best accuracy and recall - 97.4% and 93.8%.

Disease detection in pomegrenate leaves using Image processing
Developer January 2018 – May 2018

Pomegranate leaves images belonging to five categories of diseases were used to form a dataset. Used Image smoothing, Image enhancement, and Histogram equalization to pre-process images in MATLAB. Using K-means clustering, images were segmented. Mean, S.D, Smoothness, and do other properties were used as features. Classification of the image was done using Multi SVM classifier with 94.1% accuracy.
